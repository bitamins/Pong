Modules
	pong - holds the main game loop and initializes objects
	game_functions - holds the functionality of the game's objects
	game_statistics - holds the score and loop condition for the game
	settings - holds the settings for the games objects
	paddle - holds the information for creating a paddle object with pygame
	play_button - for displaying a play object on the screen
	scoreboard - for diplaying the score numbers of player 1 and player 2
	ball - holds the information for creating a ball object
